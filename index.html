CSS poziciónálás

Static

A static az alapértelmezett pozició, nem müködnek nála a right, left, top, bottom tulajdonságok

Relative

A relative engedélyezi a left, right, top, bottom tulajdonságok érvényesítését

Absolute

Az absolute is engedélyezi a left, right, top, bottom tulajdonságokat, de ő egy abszolut értéket határoz meg, és nem veszi figyelembe a többi elem helyzetét
